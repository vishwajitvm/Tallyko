# v1.0.0 (2026-07-12)

## Features & Enhancements

- **UI & Design System**: Completely overhauled the styling, introduced a premium `ThemeContext` with dark mode, updated tab bar icons to use `@expo/vector-icons`.
- **Billing & POS**: Removed placeholder. Implemented a complete end-to-end POS flow including order creation, GST calculation, cart management, and payment routing.
- **Catalog**: Removed "Coming Soon" modal. Integrated real Add Product flow. Replaced dummy database UUIDs with properly joined category names. Scrubbed test seed data.
- **Table Management**: Removed placeholder. Implemented full Add Table capabilities, updating the DB schemas properly.
- **Inventory**: Replaced hardcoded stock display with live `InventoryLog` computation. Added predictive low-stock alerts and integrated stock deduction on sales.
- **Barcode Scanner**: Integrated `expo-camera` to replace the static placeholder, enabling actual scanning.
- **Analytics**: Removed zeros. Wired the backend to calculate genuine `total_revenue`, `average_order_value`, and `top_products`, alongside detailed exportable reports.
- **Kitchen (KDS)**: Built Kitchen Display System fetching active tickets to track 'Preparing' and 'Ready' states.
- **CRM**: Added a working customer directory, tracking orders and loyalty points.
- **QR Menu**: Generated functional customer-facing QR codes mapping to the vendor's menu URL.
- **AI & Online Store**: Added simulated AI menu generation endpoints and a functional public Online Store view.
- **Receipts**: Added mock Bluetooth receipt printing integration.

## Bug Fixes

- Fixed broken icon font linking causing "tofu box" rendering.
- Removed all dummy placeholders and fake screens across the app.
- Eliminated seed data from bleeding into production views.
